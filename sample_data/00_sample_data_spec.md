- **業種**: 塗料・化学製品製造業

## 7. 生成スクリプト

### 7.1 実行順序

```bash
# 1. マスタデータ生成
python scripts/generate_masters.py

# 2. 通常仕訳生成
python scripts/generate_journals.py --year 2024

# 3. 不正・異常パターン埋め込み
python scripts/embed_anomalies.py

# 4. 集計・検証
python scripts/validate_data.py

# 5. エクスポート
python scripts/export_data.py --format csv,parquet
```

### 7.2 設定パラメータ

```yaml
# config/sample_data_config.yaml
company:
  name: "グローバル塗料株式会社"
  code: "GP001"
  fiscal_year: 2024
  fiscal_start: "2023-04-01"
  fiscal_end: "2024-03-31"

scale:
  annual_revenue: 588800000000  # 5,888億円
  total_assets: 750700000000    # 7,507億円
  journal_count: 450000         # 年間仕訳件数

fraud_patterns:
  enabled: true
  total_count: 1100             # 不正パターン総数

anomaly_patterns:
  enabled: true
  total_count: 3900             # 異常パターン総数

random_seed: 42                  # 再現性確保
```

## 8. 検証基準

### 8.1 データ整合性

| チェック項目 | 基準 |
|-------------|------|
| 貸借一致 | 全仕訳で借方合計＝貸方合計 |
| 期間整合 | 全仕訳が会計期間内 |
| 科目存在 | 全科目がマスタに存在 |
| 残高連続 | 前月繰越＋当月増減＝当月残高 |

### 8.2 不正検出率目標

| 検出手法 | 目標検出率 | 許容誤検出率 |
|----------|-----------|-------------|
| ルールベース | 95%以上 | 5%以下 |
| ML検出 | 80%以上 | 10%以下 |
| 複合分析 | 98%以上 | 3%以下 |
